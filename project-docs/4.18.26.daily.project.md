### Reindex definition of done (enforced going forward)

A reindex is not complete until both of these pass:
1. Log file ends with `DONE`
2. `curl "http://localhost:42110/api/search?q=test&n=5&t=all"` returns ≥5 diverse results from the VM

"Batch 1 OK" or "process launched" do not count. Run both checks before declaring success or proceeding to benchmark.
