## What This Says

- Query classifier tuning should come first.
- The classifier currently has only a few strong buckets, so many benchmark prompts fall back to `factual`.
- Source-aware ranking should be the second lever, especially for queries that mention `Claude`, `ChatGPT`, `Codex`, or `Jules`.
- The source inference fix removed the `unknown` leakage, so the next gains should come from intent routing and reranking rather than corpus repair.

## Recommended Tuning Targets

- Add temporal triggers for phrases like `around the time`, `recently`, `what was I doing`, and `what have I been working on`.
- Add decision / schema / meta-work triggers for phrases like `what did I decide`, `schema`, `taxonomy`, `prompts`, `rubric`, and `evaluation`.
- Add synthesis triggers for `what notes should I use`, `what docs should I use`, and `working together`.
- Add source-aware boosts when the query names a source family explicitly and the result source matches that family.
