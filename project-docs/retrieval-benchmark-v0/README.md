# Retrieval Benchmark v0

This folder is the working set for the first retrieval benchmark pass.

Structure:

- `Query/` — the actual query set for v0
- `Query typing, post-retrieval grouping and expansion/` — typing and retrieval-shaping criteria
- `Harness evaluation/` — scoring and review rubric for each run

Rules:

- Keep v0 small and readable.
- Prefer clear judgments over exhaustive scoring.
- Record whether the failure is query typing, retrieval, grouping, or corpus quality.
